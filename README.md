## Install and use

- Get the project code

```bash
git clone https://github.com/CharleeWa/vue3-vant-mobile.git
```

- Installation dependencies

```bash
cd vue3-vant-mobile

# with pnpm
pnpm install

```

- run

```bash
pnpm dev
```

- build

```bash
pnpm build
```
